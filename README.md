1. a class is a Blueprint of an object while an Object is a real instance created from the class blueprint.
   
2. A constructor is a special method that runs when you create an object. We use them to  automatically initialize an object's properties as soon as it is created.

3. Static methods belong to the Class while non-Static methods belon to the Object.

4. Variable scope is where a variable can be accessed in your codes.

5. when the local variable is accessed outside its scope you get a compiler error, because the local variable can only be accessed inside the method it was created in. Once outside, it dies.
